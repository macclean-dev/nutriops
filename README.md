# NutriOPS

Base inicial do SaaS multi-tenant para gestão digital de POPs, planilhas de controle e auditoria nutricional.

## Rodar localmente

Abra um servidor estático na pasta do projeto. Exemplo:

```bash
python3 -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```

## O que já existe

- estrutura visual multi-tenant
- dashboard executivo
- seleção de tenant, loja e módulo
- visão de POPs, auditoria e alertas
- base pronta para substituir os mocks por Supabase depois
